Queue Data Structure - Python
=============================

Overview
--------
Queue is a linear data structure that follows the
First In First Out (FIFO) principle.

The first element inserted is the first one to be removed.

Common Operations
-----------------
- Enqueue: Insert an element
- Dequeue: Remove an element
- Front: View the front element
- isEmpty: Check if queue is empty

Time Complexity
---------------
Enqueue: O(1)
Dequeue: O(1)

Space Complexity
----------------
O(n)

Use Cases
---------
- CPU scheduling
- Task scheduling
- Breadth First Search (BFS)
- Printer queues
